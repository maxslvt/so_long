## How to install

Clone the project :

```bash
gcl https://github.com/maxslvt/so_long
```

Go to the directory :

```bash
cd so_long
```

Create the executable with :

```bash
make
```

## ENJOY!

Execute like this (maps are stored in ./maps/ and you can easily create more):

```bash
./so_long [map]
```

Move with w,a,s,d catch all the collectibles and go to the exit!
